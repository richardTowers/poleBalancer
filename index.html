The Pole Balancing Problem

The pole balancing problem is a classic in control theory. The case I've set up here is very simple, push the cart left or right in order to balance the pole. Plan is to get you a little textarea where you can write some code to balance the pole.

Instructions

Click start to get going. Press left and right (on the keyboard or the buttons) to control the cart. Try to keep the poll upright for as long as possible.
